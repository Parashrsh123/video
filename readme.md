## Installation

### First setup .env file in the backend folder

```sh
PORT = <PORT NUMBER>
JWT_SECRET = <SECRET_KEY>
DB_NAME = <DATABASE_NAME>
```

### Install the dependencies and start the backend server.

```sh
cd backend
npm i
npm run dev
```
or
```sh
cd backend
yarn add
yarrn dev
```
### Install the dependencies and start the frontend application.

```sh
cd frontend
npm i
npm run start
```
or
```sh
cd backend
yarn add
yarrn start
```
